# Remakepires - A Decompiled Copy of the Vampires SMP Plugin
###### Credit to POW Creation for the original plugin.

This plugin is a recreation of the Vampires SMP plugin from POW Creation's Vampires series. <br/>
By fixing up decompilation errors and documenting the functions, this repo attempts to give plugin modders the source code to modify at their leisure.

If you would like to add a new feature to the plugin, create a branch off of **Master**. The **Master** branch will remain a stable version of the configurable Vampires plugin, with custom additions but no major game-changing features that cannot be turned off.

##### The Current Plugin branch: POW Plugin Updates
This branch is to remain as close a copy to the original plugin as possible; no optimizations or refinements may be added that change the file or function structure of the code.<br/>
The branch **Master** is where optimizations and program improvements to the basic features will be made. If you aren't interested in using this preserved version, then begin your plugin modifications from there instead.
