# Remakepires - A Decompiled Copy of the Vampires SMP Plugin
###### Credit to POW Creation for the original plugin.

This plugin is a recreation of the Vampires SMP plugin from POW Creation's Vampires series. <br/>
By fixing up decompilation errors and documenting the functions, this repo attempts to give plugin modders the source code to modify at their leisure.

If you would like to add a new feature to the plugin, create a branch off of **Master**. The **Master** branch will remain a copy of POW's current plugin version (or whatever I'm managed to update it to).

##### The Current Plugin branch: Master
This branch is the backup instance of the VSMP plugin remake. The **Master** branch needs to remain as close a copy to the original plugin as possible; no optimizations or refinements may be added that change the file or function structure of the code.<br/>
The branch **Documentation and Optimization** is where optimizations and program improvements to the basic features will be made. If you aren't interested in using this preserved version, then begin your plugin modifications from there instead.

##### The current VSMP Plugin version updated: 1.0.6
