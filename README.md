# Remakepires - A Decompiled Copy of the Vampires SMP Plugin
###### Credit to POW Creation for the original plugin.

This plugin is a recreation of the Vampires SMP plugin from POW Creation's Vampires series. <br/>
By fixing up decompilation errors and documenting the functions, this repo attempts to give plugin modders the source code to modify at their leisure.

If you would like to add a new feature to the plugin, create a branch off of Master. The Master branch will remain a copy of POW's current plugin version (or whatever I'm managed to update it to).

##### The Current Plugin branch: Documentation and Optimization
This branch is the working location for making updates that improve the experience of working within the VampireSMP plugin without changing the final functionality of the plugin itself. This includes things like creating new methods, reworking existing classes, and adding more options to the config. Anything goes, as long as from the player's perspective nothing appears to have changed.
